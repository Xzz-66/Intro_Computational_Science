# Introduction to Computational Science Projects
This repository contains two course projects completed for **Introduction to Computational Science** at the **University of Amsterdam (2025–2026)**. Both projects study infectious-disease dynamics, but from different modeling perspectives:

- **Project 1:** deterministic compartmental modeling with ordinary differential equations (ODEs)
- **Project 2:** stochastic and spatial epidemic modeling, including Gillespie simulation and network-based SIR dynamics

Together, these projects explore how epidemic behavior changes when we move from continuous deterministic systems to stochastic simulations and structured contact networks.
